# 🚀Real-Time Object Detection using YOLOv8
This project implements real-time object detection using YOLOv8 and OpenCV.  
It detects and classifies objects from a live webcam feed with bounding boxes and confidence scores.

# Features
- Real-time object detection
- Bounding box labeling
- Confidence score display
- FPS monitoring
- Webcam integration

# Technologies Used
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- NumPy

# Installation
1. Clone the repository:
git clone https://github.com/alfiyaprofessional-016/YOLO-Object-Detection

2. Navigate to the folder:
cd YOLO-Object-Detection

3. Install dependencies:
pip install -r requirements.txt

# Run the Project
python main.py
